Servicio REST
Un servicio REST es una aplicacion web que cumple una serie de patrones determinados
- Verbos HTTP (protocolo de peticion)
- Get (obtiene informacion de un servidor)
- Post (Envio datos al servidor)
- Put (Actualizar algo que ya exista con todos los datos que ya existen mas los nuevos)
- PATCH (Actualiza unicamente los datos que han cambiado)
- Delete (Eliminar un recurso que ya exista)
